# Contribution Guidelines

Thank you for your interest in contributing to our project! Here are the simple steps to get started.

## 1. Fork Repository
- Fork the main project to your GitHub account.

## 2. Clone and Create New Branches
```bash
git clone https://github.com/SweetLove1643/QuickQuiz.git
git checkout -b <your-branch-name>
```
## 3. Making Changes
- Edit the code or add new features.
- Make sure the code is clean and follows the basic rules (if applicable).

## 4. Commit and Push
```bash
git add .
git commit -m "Briefly describe your changes"
git push origin <your-branch-name>
```

## 5. Submit Pull Request (PR)
- Go to the repository page on GitHub and create a PR from your branch to the 'main' (or 'master') branch of the original project.
- Clearly describe the changes you've made.

Thank you for your contribution!
